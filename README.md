# Fleet Equipment Inventory Data Cleaning Project

## 📌 Project Overview

This project focuses on cleaning and preparing raw fleet equipment inventory data using Excel. The goal is to ensure data accuracy, consistency, and usability for analysis.

---

## 🧠 Objective

To transform raw CSV data into a clean and structured dataset by identifying and correcting data quality issues such as duplicates, missing values, inconsistent formatting, and incorrect department entries.

---

## 📁 Dataset

- Raw file: Montgomery_Fleet_Equipment_Inventory_FA_PART_1_START.csv
- Final cleaned file: Montgomery_Fleet_Equipment_Inventory_FA_PART_1_END.xlsx

---

## 🧹 Data Cleaning Steps

- Converted CSV to Excel (.xlsx format)
- Adjusted column widths for readability
- Removed empty rows
- Removed duplicate records
- Corrected spelling errors
- Removed extra whitespace using Find & Replace
- Standardized department names using Flash Fill
- Removed unnecessary columns
